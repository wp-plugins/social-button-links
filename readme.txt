Allows you to create a shortcode of social button links.

== Description ==

This plugin allow you to create your own social button links combination shortcode that you can use anywhere on your WordPress site. 

= Features =

 - Enter social button link on settings->Social Button Links and generate the shortcode
 - Place the shortcode to your WordPress site

== Installation ==

unzip archive to wp-content/plugins directory, and activate it in Plugins menu in wp-admin
Then go to settings->Social Button Links and enter the social button links that you wants and generate the shortcode.

== Frequently Asked Questions ==

= how to use this plugin =

Go to Wordpress settings -> Social Button Links and enter the social button links that you wants and generate the shortcode. Now just copy and paste it anywhere in your WordPress site.
